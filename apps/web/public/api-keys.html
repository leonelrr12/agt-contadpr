📖 Cómo usar tu API Key

Incluye la key en el header Authorization de tus peticiones HTTP.

cURL
curl -X POST https://contador507.com/api/transactions \ -H "Authorization: Bearer sk_live_TU_API_KEY" \ -H "Content-Type: application/json" \ -d '{"type":"GASTO","amount":50,"description":"Gasolina","concept":"Combustible","date":"2026-07-15"}'
JavaScript / Node.js
fetch('https://contador507.com/api/transactions', { method: 'POST', headers: { 'Authorization': 'Bearer sk_live_TU_API_KEY', 'Content-Type': 'application/json' }, body: JSON.stringify({ type: 'GASTO', amount: 50, description: 'Gasolina', concept: 'Combustible', date: '2026-07-15' }) })
Zapier / Make.com
